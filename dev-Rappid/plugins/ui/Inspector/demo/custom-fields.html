Try to click an element or link option tool to bring up an Inspector.

This demo shows how to integrate your own special fields into the ui.Inspector. We show how to integrate the select2 extension to the classical HTML <select> input and some custom buttons.

Try to call r.attr('text/style/text-decoration', 'underline') in the browswer console. You should see the change reflected in the Inspector. This demonstrates the two-way data binding that Inspector creates.
